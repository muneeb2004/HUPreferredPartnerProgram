## Workflow Rules
- No phase advances to **Implementation** without a `PASS` or `PASS WITH RECOMMENDATIONS` on its Architecture Review.
- No phase is marked **Complete** without a `PASS` on its Implementation Review.
- Development must strictly resume at the **next uncompleted phase** unless a formal Repository Realignment is requested.
