# AI Governance Framework

This `.ai/` directory contains machine-readable metadata, context, rules, checklists, and templates to prevent architectural drift when multiple AI agents work on the repository.

## Contents
- `context/`: Core business and architectural context.
- `rules/`: Engineering, design, and performance standards.
- `skills/`: Reusable AI personas.
- `checklists/`: Quality gates and review steps.
- `templates/`: Prompt templates and RFC forms.
- `workflows/`: Standard operating procedures.
