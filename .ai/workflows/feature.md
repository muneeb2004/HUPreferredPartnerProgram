# Feature Workflow

1. Read relevant `docs/` and `.ai/rules/`.
2. Create branch: `feature/short-description`.
3. Implement logic strictly following constraints.
4. Add tests.
5. Ensure `pnpm lint`, `pnpm typecheck`, `pnpm test` pass.
6. Submit PR using `.github/PULL_REQUEST_TEMPLATE.md`.
7. Await Reviewer approval.
