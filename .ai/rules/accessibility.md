# Accessibility Standards

1. **WCAG 2.2 AA**: All UIs must conform to this standard.
2. **Semantic HTML**: Use `<button>`, `<nav>`, `<main>`, `<article>`, etc.
3. **Keyboard Nav**: Every interactive element must be reachable and actionable via keyboard.
4. **Contrast**: Minimum 4.5:1 contrast ratio for normal text.
