# Performance Standards

1. **Core Web Vitals**:
   - LCP < 2.5s
   - FID < 100ms
   - CLS < 0.1
2. **Bundles**: Keep first-load JS under 150kb (gzipped).
3. **Images**: Next.js Image component for WebP/AVIF formats.
4. **Caching**: Utilize ISR for partner pages and SWR/React Query for dynamic data.
